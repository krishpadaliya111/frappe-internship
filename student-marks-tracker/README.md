# Student Marks Tracker

## Project Description

This is a simple Command Line Interface (CLI) project developed using Python.

The application allows users to:

- Add student records
- View all student records
- Search a student by name
- Store data permanently using a text file

## Concepts Used

- Variables
- Functions
- Loops
- File Handling
- Exception Handling
- Conditionals
- CLI

## Files

student_tracker.py → Main Program

students.txt → Stores student records

README.md → Project Documentation
